## 🔄 現在のステップ

### ステップ1: GitHub Desktopをインストール

**今すぐやること:**

1. ブラウザで以下のURLを開いてください：
   ```
   https://desktop.github.com/
   ```

2. 「Download for Windows」ボタンをクリック

3. ダウンロードした `GitHubDesktopSetup.exe` を実行

4. インストールが完了したら、GitHub Desktopが自動的に起動します

5. 「Sign in to GitHub.com」をクリック

6. ブラウザでGitHubアカウントにログイン
   - アカウントがない場合は「Create an account」で作成
   - メールアドレスとパスワードを設定

7. 「Authorize desktop」をクリック

8. GitHub Desktopに戻る
