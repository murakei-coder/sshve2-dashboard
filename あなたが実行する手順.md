## ❓ トラブルシューティング

### Q1: ビルドが失敗する

**エラー**: `ModuleNotFoundError: No module named 'flask'`

**解決策**:
```bash
# requirements_production.txt を確認
# 以下が含まれているか確認
flask
pandas
gunicorn
```

### Q2: アプリが起動しない

**エラー**: `Application failed to start`

**解決策**:
1. Amplifyコンソールで「ビルド設定」を開く
2. 以下を追加：

```yaml
version: 1
backend:
  phases:
    build:
      commands:
        - pip3 install -r requirements_production.txt
frontend:
  phases:
    build:
      commands:
        - gunicorn --bind 0.0.0.0:$PORT wsgi:application
```

### Q3: データが読み込まれない

**原因**: データファイルがGitにプッシュされていない

**解決策**:
```bash
# データファイルを追加
git add sshve2_data_new_suppression_20260318_222020.json
git commit -m "Add data file"
git push
```

Amplifyが自動的に再デプロイします。

### Q4: GitHubに接続できない

**解決策**:
1. GitHubでPersonal Access Tokenを作成
2. Amplifyコンソールで「GitHub接続」を再試行
3. トークンを入力
