# SSHVE2ダッシュボード - パスワード認証機能 実装完了

## 実装概要

SSHVE2ダッシュボードにクライアントサイドのパスワード認証機能を実装しました。

## 実装内容

### 1. パスワードハッシュ生成ツール
- **ファイル**: `generate_password_hash.html`
- **機能**: SHA-256ハッシュ（1000回イテレーション）を使用してパスワードハッシュを生成
- **使い方**: ブラウザで開いて、パスワードを入力し、「ハッシュを生成」をクリック

### 2. 認証システム（index.html）
以下の機能を実装しました：

#### 認証機能
- パスワード入力オーバーレイ
- SHA-256ハッシュによるパスワード検証
- セッションストレージによる認証状態管理
- ローカルストレージによる永続的認証（Remember Me機能）
- 30日間の認証有効期限
- ログアウト機能

#### セキュリティ機能
- パスワードのハッシュ化（1000回イテレーション）
- ソルト付きハッシュ
- プレーンテキストパスワードの非表示
- コンソールへのパスワードログ出力防止
- 古いブラウザ向けフォールバック機能

#### UI機能
- Enterキーでのログイン
- エラーメッセージ表示
- 入力フィールドの自動フォーカス
- Remember Meチェックボックス
- ログアウトボタン（認証後に表示）

## デフォルトパスワード

**パスワード**: `sshve2024`

現在は一時的にプレーンテキスト比較を使用していますが、本番環境では以下の手順でハッシュを生成してください。

## パスワードハッシュの生成手順（推奨）

1. `generate_password_hash.html` をブラウザで開く
2. パスワード "sshve2024" を入力
3. 「ハッシュを生成」をクリック
4. 生成されたハッシュをコピー
5. `index.html` を開き、以下の行を見つける：
   ```javascript
   passwordHash: 'TEMP_USE_GENERATE_TOOL',
   ```
6. コピーしたハッシュに置き換える：
   ```javascript
   passwordHash: '生成されたハッシュ',
   ```
7. 保存してコミット

## 動作確認

### ローカルテスト
1. `index.html` をブラウザで開く
2. パスワード入力画面が表示されることを確認
3. パスワード "sshve2024" を入力してログイン
4. ダッシュボードが表示されることを確認
5. ページをリロードしても認証状態が保持されることを確認
6. 「ログアウト」ボタンをクリックして再度パスワード画面が表示されることを確認

### Remember Me機能のテスト
1. パスワード入力画面で「ログイン状態を保持する（30日間）」にチェック
2. ログイン
3. ブラウザを完全に閉じる
4. 再度ブラウザを開いて `index.html` にアクセス
5. パスワード入力なしでダッシュボードが表示されることを確認


1-2分後にGitHub Pagesに反映されます。

## パスワードの変更方法

1. `generate_password_hash.html` で新しいパスワードのハッシュを生成
2. `index.html` の `AUTH_CONFIG.passwordHash` を更新
3. GitHubにコミット＆プッシュ
4. ユーザーに新しいパスワードを通知

## 注意事項

### セキュリティ
- これはクライアントサイド認証のため、決定的なセキュリティ対策ではありません
- 内部ダッシュボードや低感度データ向けです
- 機密情報や個人情報には使用しないでください

### ブラウザ互換性
- Chrome 37+
- Firefox 34+
- Safari 11+
- Edge 12+
- 古いブラウザではフォールバック機能が動作します

## トラブルシューティング

### パスワードを忘れた場合
管理者が `index.html` の `AUTH_CONFIG.passwordHash` を確認するか、新しいハッシュを生成してください。

### ログインできない場合
1. ブラウザのコンソールを開く（F12）
2. エラーメッセージを確認
3. ブラウザのCookieとストレージが有効か確認
4. JavaScriptが有効か確認

### 認証状態がクリアされる場合
- セッションストレージ: ブラウザを閉じるとクリアされます
- ローカルストレージ: 30日後に自動的に期限切れになります
